# Fettle Backend

Django-based system for managing and analyzing hospital-to-patient communications, automating inbound and outbound calls using AI (Vapi.ai, LiveKit, OpenAI).

## Features
- Inbound and Outbound calling orchestration.
- Transcript, summary, and sentiment generation.
- Data Processing and PDF Reporting.
- See `docs/DEVOPS.md` and `GEMINI.md` for technical lessons learned, SIP configuration, and deployment details.
